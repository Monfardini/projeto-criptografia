Criptografia — César, Vigenère, Hill, OTP e AES

Dicas de uso

  • César: chave é um número (ex: 3). Works with letters only; mantém maiúsculas/minúsculas.
  • Vigenère: chave é uma palavra (apenas letras).
  • Hill: chave: quatro inteiros separados por espaço, exemplo: 3 3 2 5. Texto usa apenas letras minúsculas; se ímpar, adiciona 'x'.
  • OTP: chave deve ter exatamente o mesmo tamanho que a mensagem. Para demonstração usamos XOR por caractere.
  • AES-128: modo ECB com padding PKCS#7. A chave AVISADA: 16 caracteres (128 bits). Se a chave for menor, o sistema a pad/recorta para 16 bytes. Aviso: ECB não é seguro para dados repetidos — para aplicações reais use AES-CBC/GCM com IV/nonce e autenticação.
